# Punto Ágil POS
Sistema de Punto de Venta.
## Descripción
Punto Ágil POS es un sistema web desarrollado para facilitar la administración de pequeños y medianos negocios. Permite gestionar ventas, compras, productos e inventario desde una única plataforma, optimizando procesos operativos y mejorando el control de la información.

El sistema incorpora autenticación de usuarios, monitoreo de inventario, generación de reportes financieros y visualización de estadísticas en tiempo real, proporcionando una herramienta eficiente para la toma de decisiones y la gestión diaria del negocio.

## Tecnologías utilizadas
#### Frontend
- HTML5
- CSS3
- JavaScript
#### Backend
- Node.js
- Express.js
#### Base de Datos 
- MySQL
#### Dependencias 
- mysql2 (conexión y consultas a MySQL)
- bcrypt (cifrado de contraseñas)
- SweetAlert2 (alertas y mensajes personalizados)
- Chart.js (gráficas)

## Funcionalidades 
- Registro e inicio de sesión de usuarios 
- Gestión de productos (registrar, editar y eliminar)
- Monitoreo de inventario mediante cards visuales permitiendo identificar rapidamente productos agotados o por agotarse
- Registro y procesamiento de ventas 
- Actualización automática de existencias 
- Reportes financieros del negocio
- Identificación de productos mas vendidos 
- Cálculo automático de utilidades por período
- Dashboard con indicadores y gráfica 
- Historial de compras 
- Historial de ventas
